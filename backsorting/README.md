# 🌀 Backsorting

**Backsorting** é um programa em C que implementa e compara diferentes algoritmos de ordenação (sorts).  
O usuário pode escolher qual método aplicar sobre um vetor e visualizar o tempo de execução em milissegundos.  
Ideal para aprendizado e análise de desempenho de algoritmos de ordenação em determinadas listas.

## 🔧 Funcionalidades

- Execução de alguns algoritmos de ordenação:
  - Bubble Sort
  - Insertion Sort
  - Selection Sort
  - Quick Sort
  - Merge Sort
- Medição de tempo de execução (em ms) usando `clock()`
- Modo **Benchmark**: executa todos os sorts e mostra o mais rápido

## ❓ Como utilizar
O programa espera que tenha um arquivo chamado "target.txt" no mesmo diretório. Ele vai usar a lista de números dentro desse arquivo para criar um novo com os números ordenados. ele também vai criar um arquivo com os logs de execução.
