## 📱 Instalação via Expo Go

### Passo 1: Instale o Expo Go

- **iOS**: [App Store](https://apps.apple.com/app/expo-go/id982107779)
- **Android**: [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)

### Passo 2: Escaneie o QR Code

1. Abra o **Expo Go**
2. Clique em **"Scan QR Code"** ou **"Escanear"**
3. Aponte a câmera para o QR Code acima
4. O app será carregado automaticamente

### Passo 3: Comece a Jogar!

Após o carregamento, você verá a tela inicial do RPG Meet.

## 🎮 Primeiro Uso

### Para o Mestre

1. Abra o app
2. Clique em **"Criar Sala"**
3. Preencha o nome da sessão (ex: "Campanha do Reino Perdido")
4. Selecione o tipo de RPG
5. Clique em **"Criar Sala"**
6. Compartilhe o código com seus jogadores

### Para Jogadores

1. Abra o app
2. Clique em **"Entrar em Sala"**
3. Insira o código fornecido pelo mestre
4. Digite seu nome de personagem
5. Clique em **"Entrar"**
6. Aproveite a sessão!

## 🔧 Requisitos Técnicos

### Mínimos

- **iOS**: 12.0 ou superior

- **Internet**: 5+ Mbps para melhor qualidade

## 🆘 Solução de Problemas

### Problema: QR Code não funciona

**Solução:**
1. Verifique se o Expo Go está instalado
2. Tente escanear novamente
3. Use o link de pré-visualização em vez disso

### Problema: App não carrega

**Solução:**
1. Verifique sua conexão de internet
2. Reinicie o Expo Go
3. Tente novamente em alguns segundos

### Problema: Câmera/Microfone não funcionam

**Solução:**
1. Verifique as permissões do app nas configurações
2. Reinicie o app
3. Verifique se outro app está usando câmera/microfone

### Problema: Conexão lenta

**Solução:**
1. Aproxime-se do roteador Wi-Fi
2. Reduza a qualidade de vídeo
3. Feche outros apps que usam internet
