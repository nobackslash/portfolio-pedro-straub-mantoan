# RPG Meet - Guia de Uso

## 🎲 Bem-vindo ao RPG Meet!

O RPG Meet é um aplicativo de videoconferência especializado para sessões de RPG de mesa, integrando o Jitsi Meet com um sistema interativo de rolagem de dados.

---

## 🚀 Como Começar

### Instalação

1. **Escanear QR Code**: Use o Expo Go (disponível na App Store ou Google Play) para escanear o QR Code fornecido
2. **Ou acessar via link**: Abra o link de pré-visualização em seu navegador

### Primeiros Passos

Ao abrir o app, você verá a tela inicial com duas opções principais:

- **Criar Sala**: Para iniciar uma nova sessão de RPG
- **Entrar em Sala**: Para participar de uma sessão existente

---

## 📋 Fluxo Principal

### 1. Criar uma Sala

1. Clique em **"Criar Sala"**
2. Preencha o **Nome da Sala** (ex: "Aventura na Floresta Escura")
3. Selecione o **Tipo de RPG**:
   - D&D 5e
   - Pathfinder
   - Call of Cthulhu
   - World of Darkness
   - Genérico
4. Clique em **"Criar Sala"**
5. Você será redirecionado para a tela de videoconferência

### 2. Compartilhar a Sala

Após criar a sala, você receberá um **código da sala**. Compartilhe este código com seus amigos para que eles possam entrar.

### 3. Entrar em uma Sala

1. Clique em **"Entrar em Sala"** na tela inicial
2. Insira o **Código da Sala** fornecido pelo mestre
3. Digite seu **Nome de Personagem** (ex: "Aragorn", "Gandalf")
4. Clique em **"Entrar"**
5. Você será conectado à sessão

---

## 🎲 Sistema de Rolagem de Dados

### Tipos de Dados Disponíveis

- **d4**: Dado de 4 faces
- **d6**: Dado de 6 faces (padrão)
- **d8**: Dado de 8 faces
- **d10**: Dado de 10 faces
- **d12**: Dado de 12 faces
- **d20**: Dado de 20 faces (D&D padrão)
- **d100**: Dado de 100 faces (percentual)

### Como Rolar Dados

1. **Selecione o tipo de dado**: Toque nos botões na parte inferior da tela
2. **Defina a quantidade**: Use os botões **+** e **-** para aumentar ou diminuir (máximo 10 dados)
3. **Role os dados**: Clique no botão **"Rolar Dados"**
4. **Veja o resultado**: O resultado aparecerá em destaque na tela
5. **Histórico**: Os últimos 5 resultados aparecem abaixo do botão de rolar

### Exemplo de Rolagem

- **2d20**: Dois dados de 20 faces
- **Resultado**: 18 + 15 = 33

## 🎥 Videoconferência

O RPG Meet usa o **Jitsi Meet** para videoconferência. Você pode:

- **Ativar/desativar câmera**: Controles no Jitsi Meet
- **Ativar/desativar microfone**: Controles no Jitsi Meet
- **Compartilhar tela**: Disponível no Jitsi Meet
- **Chat**: Disponível no Jitsi Meet

### Painel de Dados

O painel de dados fica na parte inferior da tela. Você pode:

- **Minimizar**: Clique no botão **✕** para minimizar
- **Expandir**: Clique no botão **🎲** flutuante para expandir novamente

## 💡 Dicas e Truques

### Para Mestres

1. **Prepare a sala com antecedência**: Crie a sala alguns minutos antes da sessão começar
2. **Teste o áudio/vídeo**: Verifique se câmera e microfone estão funcionando
3. **Compartilhe o código**: Envie o código da sala por chat ou email para seus jogadores
4. **Use dados para eventos aleatórios**: Role dados públicos para manter a tensão

### Para Jogadores

1. **Chegue cedo**: Conecte-se alguns minutos antes da sessão começar
2. **Teste sua conexão**: Verifique se sua internet está estável
3. **Prepare seu personagem**: Tenha suas fichas de personagem à mão
4. **Comunique-se**: Use o chat ou voz para interagir com o grupo

## ⚙️ Configurações Técnicas

### Requisitos

- **Conexão de internet**: Banda larga recomendada (mínimo 2 Mbps)
- **Câmera e microfone**: Necessários para videoconferência
- **Navegador moderno**: Para acessar via web
- **Expo Go**: Para acessar via QR Code

### Permissões

O app solicitará permissão para:

- **Câmera**: Para videoconferência
- **Microfone**: Para áudio
- **Armazenamento**: Para armazenar dados locais

## 🐛 Solução de Problemas

### Problema: Não consigo conectar à sala

**Solução:**
- Verifique se o código da sala está correto
- Verifique sua conexão de internet
- Tente criar uma nova sala

### Problema: Câmera/microfone não funcionam

**Solução:**
- Verifique as permissões do app
- Reinicie o app
- Verifique se outro app está usando câmera/microfone

### Problema: Dados não aparecem

**Solução:**
- Verifique se o painel de dados está expandido
- Tente rolar novamente
- Reinicie o app

### Problema: Conexão lenta

**Solução:**
- Verifique sua conexão de internet
- Reduza a qualidade de vídeo no Jitsi Meet
- Feche outros apps que usam internet
